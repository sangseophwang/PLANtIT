## ⚙️ 주요 기능

1. 자체 로그인 / 로그아웃 / 회원가입, 소셜 로그인
2. 커뮤니티 (CRUD 기능)
3. 질병 도감 (검색 기능 포함)
4. Swiper를 활용한 원페이지 슬라이드
5. 크로스 브라우징
